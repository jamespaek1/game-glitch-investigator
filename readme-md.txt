# 🎯 Game Glitch Investigator

A number-guessing game built with Python and Streamlit — **but it's full of bugs!**
Your mission is to find, explain, and fix every glitch using AI-assisted debugging.

## 🚀 Getting Started

```bash
# 1. Clone your fork
git clone https://github.com/YOUR-USERNAME/game-glitch-investigator.git
cd game-glitch-investigator

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the game
python -m streamlit run app.py

# 4. Run the test suite
pytest
```
